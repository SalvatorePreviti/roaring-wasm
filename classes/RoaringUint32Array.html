Class RoaringUint32Array

Array of unsigned 32 bit integers allocted directly in roaring library WASM memory. Note: Memory is not garbage collected, you are responsible to free the allocated memory calling "dispose" method.

constructor

  • new RoaringUint32Array(lengthOrArray, _pointer?): RoaringUint32Array

  • Allocates an array in the roaring WASM heap.

    Note: Memory is not garbage collected, you are responsible to free the allocated memory calling "dispose" method.

    If the parameter is a number, it creates a new uninitialized array of the given length. If the parameter is an Iterable, it creates a copy of the given iterable.

    Parameters

    • lengthOrArray: number | Iterable<number>

      Length of the array to allocate or the array to copy

    • Optional _pointer: number

    Returns RoaringUint32Array
